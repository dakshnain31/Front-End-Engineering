Artificial Intelligence & Machine Learning

What is AI?

Artificial Intelligence (AI) refers to machines that can perform tasks that normally require human intelligence. These tasks include learning, problem solving, language understanding, and pattern recognition.

What is Machine Learning?

Machine Learning (ML) is a sub-field of AI where machines learn from data and improve over time without being explicitly programmed. Example: recommendation systems, spam detection, image classification.

Types of Machine Learning

✔ Supervised Learning – model learns from labeled data
✔ Unsupervised Learning – finds hidden patterns in data
✔ Reinforcement Learning – learns by rewards and penalties

Real-World Applications

- Chatbots and AI assistants
- Self-driving cars
- Face recognition
- Medical diagnosis
- Language translation
- Stock prediction
